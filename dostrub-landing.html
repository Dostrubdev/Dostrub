Interface
Roue Radiale

Sélection en
un coup de poignet

Alt + Clic gauche Maintenez le raccourci, la roue s'ouvre sous votre curseur. Glissez vers le personnage voulu, relâchez.
Icônes par classe Chaque tranche affiche l'icône de classe et le nom du personnage. Reconnaissance visuelle immédiate.
Retour haptique sonore Effets audio au survol et à la sélection. Volume ajustable directement dans les paramètres.
Personnage actif mis en valeur La fenêtre actuellement au premier plan est mise en évidence dans la roue pour ne jamais se perdre.
Installation

En route en 3 étapes

01
Télécharger

Récupère la dernière version Dostrub.exe depuis la page Releases sur GitHub.

02
Lancer en Admin

Exécute le fichier. Dostrub demande les droits administrateur pour interagir avec les fenêtres de jeu.

03
Calibrer

Dans l'onglet Paramètres → Calibrages, configure tes zones de clic (Zaap, Inventaire). C'est fait !

Sécurité : Dostrub fonctionne uniquement via la détection de pixels et la simulation d'entrées clavier/souris standard. Il n'injecte aucun code dans le client Dofus.
