# ≡ Dostrub ≡
Site internet : https://fromzerotosec.github.io/Dostrub

Dostrub est un organisateur de fenêtres et un gestionnaire de macros pour Dofus, conçu pour faciliter le multicompte de manière fluide et intuitive.

## 🚀 Fonctionnalités Principales

- **Focus Multi-perso** : Changez de fenêtre instantanément via des raccourcis personnalisables ou une roue de sélection radiale (`Alt + Clic`).
- **Sync Click** : Réduisez vos actions répétitives en synchronisant vos clics (gauche/droit) sur tous vos comptes.
- **Auto-Zaap** : Une macro intelligente qui ouvre le Havre-Sac et vous téléporte au Zaap configuré pour chaque personnage.
- **Auto-Groupe** : Invitez toute votre équipe en un seul clic ou via un raccourci clavier.
- **Auto-Échange (Scanner)** : Validez automatiquement les échanges entre vos personnages en détectant les notifications en jeu.
- **Overlay Flottant** : Une barre d'outils discrète et déplaçable au-dessus du jeu pour accéder rapidement à vos fonctions vitales.
- **Auto-Update** : Le logiciel se met à jour automatiquement via GitHub pour vous garantir la dernière version.

## ⚙️ Installation & Utilisation

1. **Lancement** : Exécutez `Dostrub.exe`. Le logiciel demandera les droits d'administrateur pour interagir avec le jeu.
2. **Calibrage** : Lors de la première utilisation, rendez-vous dans l'onglet **Paramètres** et **Calibrages** pour configurer vos zones de clic (Zaap, Inventaire, etc.).
3. **Raccourcis** : Configurez vos touches (Précédent, Suivant, Leader) dans l'onglet **Binds & Options**.

## 🛠️ Sécurité

Dostrub fonctionne principalement via la détection de pixels et la simulation d'entrées clavier/souris standard. Il n'injecte aucun code dans le client de jeu, ce qui le rend plus sûr que les bots traditionnels.
